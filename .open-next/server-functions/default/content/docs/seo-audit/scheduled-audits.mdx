## 什麼是「定期稽核 / 自動化排程」？

你可以在 **排程管理（Scheduler）** 建立週期性任務，讓系統在背景自動執行，例如：

- **SEO 審查**（整站或指定網址）
- **網站爬蟲**（更新頁面內容快取：整站 / sitemap / 指定網址）
- **內容重複檢測**（全站掃描）
- **競爭對手分析**（指定競爭對手網址）

> 功能是否可用，會依你的方案、權限與功能開關而定。

## 如何建立排程

1. 進入品牌後，前往「排程管理」頁面
2. 點擊「新增排程」
3. 選擇「任務類型」（SEO 審查 / 網站爬蟲 / 內容重複檢測 / 競爭對手分析）
4. 依任務類型選擇範圍/模式（例如：整站審查 vs 特定頁面；整站爬取 vs sitemap）
5. 設定頻率與時間（每天 / 每週 / 每月）
6. 若需要指定網址：貼上多行網址（每行一個）
7. 儲存

## 排程管理頁面會顯示什麼？

- 每個排程的 **狀態**（啟用/暫停）
- **上次執行時間** 與 **下次執行時間**
- 你可以在列表中直接 **暫停 / 啟用 / 刪除**

<Callout type="info">
  目前排程管理頁面提供的功能以「建立/管理排程」為主；不包含分數趨勢圖、警報門檻設定或報表匯出。
</Callout>
