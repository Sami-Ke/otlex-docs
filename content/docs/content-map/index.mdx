## 什麼是內容地圖？

內容地圖是 Otlex 的獨家功能，讓你以視覺化的方式了解：

- 網站頁面之間的連結關係
- 主題群集 (Topic Clusters) 分佈
- 孤島頁面 (Orphan Pages) 偵測
- 內部連結健康度

## 三種視圖

### 1. Page View（頁面視圖）

以頁面為節點，顯示頁面之間的內部連結關係。

- 節點大小代表頁面重要性
- 連線代表內部連結
- 顏色區分不同類型頁面

### 2. Topic View（主題視圖）

以主題群集為中心，顯示相關頁面的分佈。

### 3. Keyword View（關鍵字視圖）

以關鍵字為核心，顯示著陸頁面與關鍵字的對應關係。

## 如何使用

1. 進入「內容地圖」頁面
2. 選擇想要的視圖模式
3. 使用滑鼠拖曳和縮放探索
4. 點擊節點查看詳細資訊
