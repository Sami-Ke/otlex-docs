## 什麼是 AI 內容生成？

Otlex 的 AI 內容生成功能幫助你快速產出高品質的 SEO 優化內容。從主題發想、文章撰寫到多平台再利用，AI 會全程協助你完成內容行銷的工作流程。

### 核心能力

| 功能 | 說明 |
|-----|------|
| **主題建議** | AI 分析品牌定位與市場趨勢，推薦高潛力內容主題 |
| **文章生成** | 輸入主題後自動產出完整文章，包含標題、大綱、內文 |
| **品牌風格** | 學習你的寫作風格，確保內容符合品牌調性 |
| **內容再利用** | 將文章轉換為社群貼文、影片腳本等多種格式 |
| **內部連結** | 自動分析並建議最佳內部連結位置 |

## 內容生成流程

```
主題探索 → 文章生成 → 編輯優化 → 內容再利用 → 發布排程
```

### 步驟 1：主題探索

在「主題探索」功能中，你可以：

**AI 推薦主題**
- 根據品牌定位和目標關鍵字推薦主題
- 分析競品內容找出內容缺口
- 評估每個主題的搜尋量和競爭度

**手動建立主題**
- 自訂主題名稱和目標關鍵字
- 設定主題優先級和狀態
- 管理主題庫

### 步驟 2：文章生成

選擇主題後，AI 會生成完整的文章草稿：

- **SEO 優化標題**：吸引點擊且包含關鍵字
- **Meta Description**：符合搜尋引擎顯示規範
- **結構化大綱**：H2、H3 層級清晰
- **完整內文**：自然融入關鍵字的高品質內容
- **內部連結建議**：標示適合插入連結的位置

### 步驟 3：編輯優化

在內建編輯器中精修內容：

- **AI 重寫**：選取段落請 AI 改寫、精簡或擴充
- **AI 建議**：請 AI 給出改進建議
- **SEO 檢查**：即時顯示 SEO 分數和改進提示
- **富文字編輯**：支援圖片、表格、程式碼區塊

### 步驟 4：內容再利用

一篇文章可以轉換為：

| 格式 | 適用平台 |
|-----|---------|
| 社群貼文 | Facebook、LinkedIn、Twitter、Instagram |
| 影片腳本 | YouTube、TikTok、Reels |
| Podcast 大綱 | 各 Podcast 平台 |
| Infographic 內容 | 設計師參考文案 |
| Email Newsletter | 電子報內容 |

### 步驟 5：發布排程

完成的內容可以：

- 直接發布到已連接的 CMS（WordPress、Notion）
- 設定排程自動發布
- 匯出為 Markdown 或 HTML
- 加入社群排程器統一管理

## 主要功能模組

### 主題探索

AI 會從多個來源分析並推薦內容主題：

| 分析來源 | 說明 |
|---------|------|
| 品牌關鍵字 | 你設定的目標關鍵字及相關長尾詞 |
| GSC 數據 | 有曝光但還沒有內容的機會關鍵字 |
| 競品分析 | 競品有而你沒有的內容主題 |
| 趨勢數據 | 搜尋量上升中的相關主題 |

每個推薦主題會顯示：
- 預估搜尋量
- 關鍵字難度
- 內容建議方向
- 相關長尾關鍵字

### 文章生成

文章生成支援多種參數設定：

**字數範圍**

| 設定 | 字數 | 適用場景 |
|-----|------|---------|
| 簡短 | 500-800 | 新聞、快訊、簡介 |
| 標準 | 1000-1500 | 一般文章 |
| 深度 | 2000-3000 | 教學、指南 |
| 完整 | 3000+ | 終極指南、白皮書 |

**內容類型**

- 教學指南（How-to）
- 列表文章（Listicle）
- 比較評測（Comparison）
- 案例分析（Case Study）
- 觀點評論（Opinion）

**語調風格**

如果已設定品牌風格會自動套用，否則可選擇：
- 專業權威
- 友善親切
- 輕鬆幽默
- 客觀中立

### 品牌風格

品牌風格功能讓 AI 學習你的寫作特色：

- 提供 3-5 篇現有文章作為範例
- AI 分析語調、用詞、句式特徵
- 後續生成內容會自動套用風格

這確保所有 AI 生成的內容都符合品牌調性，讀起來就像你親自寫的一樣。

### 內容再利用

將已完成的文章轉換為其他格式：

- **社群貼文**：針對不同平台優化長度和調性
- **影片腳本**：包含開場、段落腳本、視覺提示
- **Podcast 大綱**：討論要點、時間估算、問題建議

## AI 額度說明

AI 功能消耗會依據方案配額：

| 操作 | 消耗額度 |
|-----|---------|
| 主題推薦 | 1 點 / 次 |
| 文章生成（標準） | 5 點 / 篇 |
| 文章生成（深度） | 10 點 / 篇 |
| AI 重寫段落 | 1 點 / 次 |
| 內容再利用 | 2 點 / 格式 |

額度每月重置，詳情請參考 [方案與定價](/docs/account-billing/plans)。

## 最佳實踐

### 獲得最佳生成品質

1. **提供清晰的主題描述**：越具體越好
2. **設定目標關鍵字**：幫助 AI 理解 SEO 目標
3. **使用品牌風格**：確保內容一致性
4. **提供參考資料**：有助於提升內容深度

### 有效率的編輯流程

1. 先看整體結構，再細修內容
2. 使用 AI 重寫處理不滿意的段落
3. 補充品牌特有的案例或數據
4. 最後檢查 SEO 分數

### 內容再利用策略

1. 不是每篇文章都適合再利用
2. 選擇資訊量豐富的長文效果最好
3. 各平台的內容需要個別調整
4. 追蹤 UTM 了解哪種格式效果最好
