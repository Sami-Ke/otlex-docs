## 什麼是品牌風格？

每個品牌都有自己獨特的「聲音」——用詞習慣、語調、句式與內容結構。Otlex 的「品牌寫作風格」會根據你已爬取的網站內容建立一份 **Style Profile**，讓 AI 在生成文案時能更貼近你的品牌調性。

| 能力 | 說明 |
|-----|------|
| **學習你的風格** | 分析已爬取的網站頁面，提取語氣、句式、用詞與結構特徵 |
| **一致性輸出** | 文章生成、改寫、社群貼文等 AI 文案會可選擇自動套用此 Profile |
| **減少編輯時間** | 生成內容更接近你的品牌語調與用詞偏好 |

## 設定品牌風格

### 步驟 1：先準備「已爬取」的頁面

品牌寫作風格是從你已爬取的網站內容中分析而來，因此請先確保品牌已完成一次網站爬取（讓系統有可分析的頁面清單）。

### 步驟 2：進入「寫作風格」分頁

1. 進入「設定」
2. 開啟「品牌設定」
3. 切換到「寫作風格」

### 步驟 3：選擇風格範本頁面（可選，但建議）

在寫作風格區塊中，你可以開啟「風格範本頁面」選擇器，從已爬取的頁面裡勾選你希望用來代表品牌風格的內容（可搜尋標題/URL）。

<Callout type="info">
  建議至少選 5 篇以上、且語氣一致的頁面作為範本，分析結果會更穩定。
</Callout>

### 步驟 4：開始分析並等待完成

點擊「開始分析寫作風格」後，系統會建立背景任務進行分析。完成後會出現「目前風格 Profile」，並顯示：

- 語氣（tone）與範例句
- 句式（平均句長、複雜度）與範例句
- 用詞偏好（preferred terms）、避免用語（avoid terms）與範例
- 內容結構（開頭/結尾/轉折模式）
- 分析信心度與來源頁面數量

## 範例選擇指南

### 理想的範本頁面

| 條件 | 原因 |
|-----|------|
| **原創/品牌自有內容** | 確保是你的真實風格 |
| **文字量充足** | 內容越完整，越容易抽出穩定特徵 |
| **語氣一致** | 避免混雜多種寫法導致風格變得模糊 |
| **近期內容** | 更能反映目前的品牌調性 |

### 範例數量建議

| 數量 | 效果 |
|-----|------|
| 1 篇 | ⚠️ 不建議，樣本太少 |
| 3 篇 | ✅ 最低建議數量 |
| 5 篇 | ✅✅ 理想數量 |
| 8+ 篇 | ✅✅✅ 最佳效果 |

### 應避免的範例

| 類型 | 問題 |
|-----|------|
| 轉載內容 | 不是你的風格 |
| 客座文章 | 風格可能不一致 |
| 翻譯內容 | 翻譯腔會影響分析 |
| 過於舊的文章 | 可能不代表目前風格 |
| 太短的內容 | 樣本不足 |
| 風格混亂的文章 | 會干擾學習 |

## 風格應用方式

### 自動應用

建立 Style Profile 後，你可以選擇在生成文案時自動套用此風格（以 App 畫面與權限為準）：

| 功能 | 應用方式 |
|-----|---------|
| 文章生成 | 生成時參考你的語氣/句式/用詞偏好 |
| AI 改寫 | 改寫時優先維持你的用詞與語調 |

<Callout type="info">
  在「目前風格 Profile」區塊中可以切換「生成文案時自動套用此風格」。
</Callout>

## 多品牌風格管理

如果你管理多個品牌（多個 Brand），每個品牌可以有獨立風格：

### 設定方式

1. 在左上角切換到目標品牌
2. 進入該品牌的「風格設定」
3. 分別設定各自的範例內容
4. 各品牌風格互相獨立

### 實際應用

```
品牌 A（B2B SaaS）
├── 語調：專業權威
├── 用詞：技術術語多
└── 風格：正式、數據導向

品牌 B（生活消費品）
├── 語調：友善親切
├── 用詞：口語化
└── 風格：輕鬆、有趣味
```

切換品牌後，AI 會自動使用該品牌的風格設定。

## 風格效果評估

### 如何判斷風格設定是否成功

生成幾篇測試文章後，檢查：

| 檢查項目 | 判斷標準 |
|---------|---------|
| 語調 | 讀起來像你自己寫的嗎？ |
| 用詞 | 有使用你習慣的詞彙嗎？ |
| 句式 | 句子長度和節奏符合期待嗎？ |
| 整體感 | 放到你的網站上違和嗎？ |

### 改善效果的方法

如果效果不理想：

1. **增加範例**：更多範例 = 更準確的學習
2. **精選範例**：只保留最能代表品牌的內容
3. **手動調整**：微調語調滑桿和關鍵詞設定
4. **重新分析**：修改後點擊「重新分析」

## 風格維護建議

品牌風格可能隨時間演變，建議定期維護：

| 頻率 | 建議動作 |
|-----|---------|
| **每季** | 檢視風格摘要是否仍然準確 |
| **發布新文章後** | 考慮將滿意的文章加入範例 |
| **品牌調整時** | 更新範例內容並重新分析 |
| **效果下降時** | 刪除不再代表品牌的舊範例 |

## 常見問題

### 設定風格要多久？

首次設定約需 10-15 分鐘（準備範例 + 等待分析）。設定完成後就會自動應用。

### 風格設定會影響已生成的文章嗎？

不會。風格設定只影響之後生成的新內容。已生成的文章保持不變。

### 可以完全移除品牌風格嗎？

可以。在風格設定頁面點擊「清除風格設定」，之後會使用預設風格。

### 不同類型文章適合不同風格怎麼辦？

可以在生成時手動選擇風格。例如新聞類內容選「客觀中立」，教學類內容使用品牌風格。
