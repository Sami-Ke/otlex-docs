## 什麼是品牌風格？

每個品牌都有自己獨特的「聲音（Voice）」——用詞習慣、語調、句式與內容結構。

Otlex 的「品牌寫作風格（Style Profile）」會分析你已爬取的網站頁面，自動提取以下特徵，讓 AI 在生成文案時更貼近你的品牌調性：

| 分析維度 | 說明 |
|---------|------|
| **語氣（Tone）** | 你的文章是專業嚴謹、親切友善、還是幽默輕鬆？ |
| **句式結構** | 平均句長、喜歡用短句還是長句、是否常用條列式 |
| **用詞偏好** | 你習慣用的特定詞彙或行業術語 |
| **避免用語** | 你不希望 AI 使用的詞彙或表達方式 |
| **內容結構** | 開頭引言的方式、段落組織、結尾 Call-to-Action 模式 |

### 品牌風格能帶來什麼好處？

| 能力 | 說明 |
|-----|------|
| **一致性輸出** | 文章生成、AI 改寫、社群貼文都更接近你的品牌語調 |
| **減少編輯時間** | 生成後不需要大幅改寫，只需微調細節 |
| **多成員協作** | 不同成員生成的內容風格保持一致 |
| **規模化生產** | 大量生成時不失去品牌識別度 |

## 設定品牌風格（步驟詳解）

### 步驟 1：先完成網站爬蟲

品牌寫作風格是從你已爬取的網站內容分析而來。

**請先確認：**
- 品牌已完成至少一次網站爬取（進入「設定 > 網站管理」或「內容地圖」頁面確認頁面數量）
- 爬取的頁面中有你的原創品牌內容（非廣告頁、政策頁等）

若尚未爬取，請先在「設定 > 網站管理」啟動爬取，等待完成後再設定風格。

### 步驟 2：進入「寫作風格」頁面

1. 進入「設定」
2. 開啟「品牌設定」
3. 切換到「寫作風格（Style Profile）」分頁

### 步驟 3：選擇風格範本頁面

這是最重要的一步。你需要從已爬取的頁面中，**選出最能代表你品牌風格的頁面**作為分析素材。

**操作方法：**
1. 點擊「選擇範本頁面」開啟頁面選擇器
2. 可以搜尋頁面標題或 URL 快速找到目標頁面
3. 勾選要作為風格範本的頁面

**選擇頁面的建議：**

| 條件 | 原因 |
|-----|------|
| **選取原創/品牌自有內容** | 確保分析的是你自己的真實風格 |
| **選文字量充足的頁面** | 內容越完整，越容易提取穩定特徵 |
| **語氣一致的頁面** | 避免混雜多種風格導致 Profile 模糊 |
| **近期撰寫的內容** | 更能反映目前的品牌調性 |
| **選 5-10 篇** | 樣本足夠多才能得出穩定結果 |

**應避免選入的頁面類型：**

| 類型 | 問題 |
|-----|------|
| 轉載的新聞稿或外部文章 | 不是你的風格 |
| 客座文章 | 風格可能與品牌不符 |
| 翻譯自其他語言的內容 | 翻譯腔會干擾分析 |
| 過於老舊（1-2 年前）的文章 | 可能不代表目前的品牌方向 |
| 太短的頁面（< 300 字） | 樣本不足，分析結果不穩定 |
| 政策頁、聯絡頁等功能性頁面 | 這些頁面的語氣通常不代表品牌個性 |

### 範本頁面數量建議

| 頁面數量 | 分析效果 |
|---------|---------|
| 1 篇 | ⚠️ 不建議，樣本太少 |
| 3 篇 | ✅ 最低建議數量，結果可能不夠穩定 |
| 5-8 篇 | ✅✅ 理想範圍，效果最佳 |
| 10+ 篇 | ✅✅✅ 最佳效果，但邊際效益遞減 |

### 步驟 4：啟動分析

選好頁面後，點擊「開始分析寫作風格」，系統會：

1. 建立背景分析任務
2. AI 讀取你選取的所有頁面內容
3. 提取語言特徵並生成 Style Profile
4. 完成後在頁面上顯示分析結果

**預計時間：** 通常需要 2-5 分鐘，取決於選取的頁面數量和內容長度。

## Style Profile 分析結果說明

分析完成後，你會看到一份詳細的品牌風格報告，包含：

### 1. 整體語氣摘要

AI 對你品牌整體語調的描述，例如：

```
「你的品牌語氣整體上是專業且務實的，
 傾向使用數據和案例佐證觀點，
 偶爾加入問句邀請讀者思考。」
```

### 2. 用詞偏好清單

系統識別出你**常用**的詞彙和**應避免**的詞彙：

| 類型 | 範例 |
|------|------|
| **偏好用語** | `優化`、`關鍵字策略`、`轉換率`、`洞察` |
| **避免用語** | `革命性`、`顛覆`、`驚豔`（過度行銷語言） |

### 3. 句式特徵

- 平均句子長度（例如：「傾向使用 15-25 字的中等長度句子」）
- 句型習慣（問句、強調句、條列式的使用頻率）
- 段落結構（段落長度、每段字數）

### 4. 內容結構模式

分析你如何開始和結束文章：

```
開頭模式：「通常以提出問題或陳述痛點作為開頭引言」
結尾模式：「傾向以行動呼籲（CTA）結尾，引導讀者採取下一步」
```

### 5. 信心度指標

| 指標 | 說明 |
|------|------|
| **分析信心度** | 基於樣本量和內容一致性的置信度（百分比） |
| **來源頁面數** | 分析使用了多少頁面 |

信心度低於 70% 建議增加範本頁面數量或選取更一致的頁面。

## 啟用品牌風格套用

分析完成後，在「目前風格 Profile」區塊可以切換「**生成文案時自動套用此風格**」。

啟用後，以下功能會自動套用你的 Style Profile：

| 功能 | 如何套用 |
|-----|---------|
| **AI 文章生成** | 語氣、句式、用詞偏好作為生成參數 |
| **選取改寫（AI Rewrite）** | 改寫時優先維持你的用詞和語調風格 |
| **社群貼文生成** | 貼文草稿符合品牌語調 |

## 多品牌風格管理

如果你管理多個品牌（多個 Brand），每個品牌可以有獨立的 Style Profile：

### 設定方式

1. 在左上角切換到目標品牌
2. 進入該品牌的「設定 > 品牌設定 > 寫作風格」
3. 分別為每個品牌選取範本頁面並分析
4. 各品牌的 Profile 完全獨立，不互相影響

### 多品牌風格範例

```
品牌 A（B2B SaaS）
├── 語調：專業、資料導向、務實
├── 偏好用語：ROI、效率、工作流程
└── 避免用語：「革命性」、「顛覆性」

品牌 B（生活時尚電商）
├── 語調：親切、感性、有個人色彩
├── 偏好用語：「你」、生活感句式、正向情緒詞
└── 避免用語：太正式的術語
```

切換品牌後，AI 會自動使用該品牌對應的 Style Profile。

## 評估風格效果與持續改善

### 如何判斷設定是否成功

生成幾篇測試文章後，對照以下檢查清單：

| 檢查項目 | 判斷問題 |
|---------|---------|
| 整體語調 | 讀起來像你自己寫的嗎？ |
| 用詞選擇 | 有使用你習慣的詞彙嗎？是否避開了你不喜歡的說法？ |
| 句子節奏 | 句子長度和節奏符合期待嗎？ |
| 整體感 | 把這篇文章放上你的網站，看起來違和嗎？ |
| 受眾匹配 | 文章的語氣適合你的目標讀者嗎？ |

### 效果不理想時的改善方法

| 問題 | 解決方式 |
|------|---------|
| 語調太通用 | 選取更具代表性的範本頁面，增加數量 |
| 某些詞彙不該出現 | 在 Profile 的「避免用語」手動加入 |
| 風格不一致 | 移除風格差異大的範本頁面後重新分析 |
| 沒有達到預期的專業度 | 選取你最滿意的幾篇文章作為範本 |
| AI 仍然用了不喜歡的說法 | 在文章生成的「自訂提示詞」中額外說明 |

## 品牌風格維護建議

品牌風格會隨時間演變，建議定期更新：

| 時機 | 建議動作 |
|------|---------|
| **每季** | 檢視 Style Profile 摘要是否仍然準確 |
| **發布滿意的文章後** | 考慮將該文章加入範本頁面清單 |
| **品牌策略調整時** | 更新範本頁面並重新分析 |
| **效果明顯下降時** | 刪除不再代表品牌的舊範本，重新分析 |
| **網站改版後** | 重新爬取並更新風格範本 |

## 常見問題

### 首次設定需要多久？

整個流程約 10-20 分鐘：
- 選取範本頁面：5-10 分鐘
- 等待 AI 分析：2-5 分鐘
- 確認結果並調整：5 分鐘

### 設定後會影響已生成的文章嗎？

不會。品牌風格設定只影響**之後**生成的新內容。已生成的文章不會改變。

### 可以完全關閉品牌風格嗎？

可以。在風格設定頁面關閉「生成文案時自動套用此風格」的開關，後續生成會使用預設風格。若想徹底清除，可點擊「清除風格設定」。

### 不同類型文章適合不同風格怎麼辦？

可以在生成時使用「自訂提示詞」覆蓋特定的風格需求：
- 需要更正式的法律類文章：提示詞加「使用正式書面語，避免口語化表達」
- 需要更輕鬆的社群文章：在貼文助手中選擇「輕鬆口語」語氣

### 沒有已爬取頁面也能設定風格嗎？

不行。品牌風格分析需要爬取的網站內容作為素材。請先完成網站爬取，再回來設定風格。
