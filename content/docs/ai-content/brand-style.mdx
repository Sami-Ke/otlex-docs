## 什麼是品牌風格？

每個品牌都有自己獨特的「聲音」——用詞習慣、語調風格、句式偏好。品牌風格功能讓 Otlex 的 AI 能夠：

| 能力 | 說明 |
|-----|------|
| **學習你的風格** | 分析現有內容，提取寫作特徵 |
| **一致性輸出** | 確保所有 AI 生成內容風格統一 |
| **品牌識別度** | 讓讀者一眼就認出是你的內容 |
| **減少編輯時間** | 生成內容更接近你的期望 |

### 風格學習的原理

AI 會分析你提供的範例內容，學習以下特徵：

- **語調特徵**：正式/親切/幽默/權威
- **用詞習慣**：常用詞彙、專業術語、口頭禪
- **句式偏好**：長句/短句比例、段落長度
- **表達方式**：直述/引導式/問句開頭
- **格式習慣**：列表使用頻率、引用方式

## 設定品牌風格

### 步驟 1：進入品牌風格設定

1. 點擊左側選單「品牌設定」
2. 切換到「風格設定」分頁
3. 點擊「設定品牌風格」開始

### 步驟 2：提供範例內容

有三種方式提供範例：

#### 方式一：輸入網址（推薦）

1. 點擊「新增範例」
2. 選擇「從網址匯入」
3. 貼上文章網址
4. 系統自動爬取並分析內容

<Callout type="info">
  建議使用你的部落格文章或官網內容，確保有足夠文字量讓 AI 學習。
</Callout>

#### 方式二：直接貼上文字

1. 點擊「新增範例」
2. 選擇「貼上內容」
3. 將文章內容貼入文字框
4. 點擊「確認」

#### 方式三：從已生成文章選取

如果之前已用 Otlex 生成過滿意的文章：

1. 點擊「從文章庫選取」
2. 勾選你最滿意的文章
3. 點擊「加入範例」

### 步驟 3：等待 AI 分析

點擊「開始分析」後，AI 會進行深度分析：

| 分析項目 | 時間估計 |
|---------|---------|
| 語調辨識 | 15-30 秒 |
| 用詞提取 | 20-40 秒 |
| 句式分析 | 15-30 秒 |
| 風格建模 | 30-60 秒 |

總計約 1-3 分鐘，視範例數量而定。

### 步驟 4：檢視與調整風格摘要

分析完成後會顯示風格報告：

#### 語調特徵

```
主要語調：友善專業（65%）
次要語調：教學引導（25%）
其他：幽默輕鬆（10%）
```

#### 關鍵用詞

系統會列出你的特色詞彙：
- 常用開頭詞：「首先」「接下來」「重點是」
- 常用連接詞：「不過」「然而」「更重要的是」
- 品牌術語：系統會識別並標記你的專業用語

#### 句式特徵

- 平均句子長度：約 25 字
- 段落平均長度：3-4 句
- 問句使用頻率：每 200 字約 1 個問句

### 步驟 5：手動微調（選用）

如果自動分析不夠準確，你可以：

| 調整項目 | 說明 |
|---------|------|
| 語調偏好 | 手動調整語調比例滑桿 |
| 必用詞彙 | 新增 AI 必須使用的詞彙 |
| 禁用詞彙 | 新增 AI 絕對不能使用的詞彙 |
| 格式偏好 | 設定列表、引用的使用頻率 |

## 範例選擇指南

### 理想的範例內容

| 條件 | 原因 |
|-----|------|
| **原創內容** | 確保是你的真實風格 |
| **1000+ 字** | 有足夠內容讓 AI 學習 |
| **代表作** | 你最滿意、最能代表品牌的文章 |
| **近期內容** | 反映目前的品牌調性 |
| **多主題** | 不同主題但風格一致 |

### 範例數量建議

| 數量 | 效果 |
|-----|------|
| 1 篇 | ⚠️ 不建議，樣本太少 |
| 3 篇 | ✅ 最低建議數量 |
| 5 篇 | ✅✅ 理想數量 |
| 8+ 篇 | ✅✅✅ 最佳效果 |

### 應避免的範例

| 類型 | 問題 |
|-----|------|
| 轉載內容 | 不是你的風格 |
| 客座文章 | 風格可能不一致 |
| 翻譯內容 | 翻譯腔會影響分析 |
| 過於舊的文章 | 可能不代表目前風格 |
| 太短的內容 | 樣本不足 |
| 風格混亂的文章 | 會干擾學習 |

## 風格應用方式

### 自動應用

設定品牌風格後，以下功能會自動套用：

| 功能 | 應用方式 |
|-----|---------|
| 文章生成 | 自動使用品牌風格 |
| AI 重寫 | 維持你的用詞和語調 |
| 內容再利用 | 轉換時保持風格一致 |
| AI 建議 | 建議用詞會參考你的習慣 |

### 手動切換風格

生成內容時，可以選擇不同風格：

| 選項 | 說明 |
|-----|------|
| **使用品牌風格** | 套用你設定的風格（預設） |
| **專業權威** | 正式、術語多、數據導向 |
| **友善親切** | 口語化、易懂、有溫度 |
| **輕鬆幽默** | 活潑、有創意、有趣味 |
| **客觀中立** | 平衡報導、引用來源 |

<Callout type="info">
  即使選擇預設風格，如果你有設定品牌風格，AI 也會適度參考你的用詞習慣。
</Callout>

## 多品牌風格管理

如果你管理多個品牌（多個 Brand），每個品牌可以有獨立風格：

### 設定方式

1. 在左上角切換到目標品牌
2. 進入該品牌的「風格設定」
3. 分別設定各自的範例內容
4. 各品牌風格互相獨立

### 實際應用

```
品牌 A（B2B SaaS）
├── 語調：專業權威
├── 用詞：技術術語多
└── 風格：正式、數據導向

品牌 B（生活消費品）
├── 語調：友善親切
├── 用詞：口語化
└── 風格：輕鬆、有趣味
```

切換品牌後，AI 會自動使用該品牌的風格設定。

## 風格效果評估

### 如何判斷風格設定是否成功

生成幾篇測試文章後，檢查：

| 檢查項目 | 判斷標準 |
|---------|---------|
| 語調 | 讀起來像你自己寫的嗎？ |
| 用詞 | 有使用你習慣的詞彙嗎？ |
| 句式 | 句子長度和節奏符合期待嗎？ |
| 整體感 | 放到你的網站上違和嗎？ |

### 改善效果的方法

如果效果不理想：

1. **增加範例**：更多範例 = 更準確的學習
2. **精選範例**：只保留最能代表品牌的內容
3. **手動調整**：微調語調滑桿和關鍵詞設定
4. **重新分析**：修改後點擊「重新分析」

## 風格維護建議

品牌風格可能隨時間演變，建議定期維護：

| 頻率 | 建議動作 |
|-----|---------|
| **每季** | 檢視風格摘要是否仍然準確 |
| **發布新文章後** | 考慮將滿意的文章加入範例 |
| **品牌調整時** | 更新範例內容並重新分析 |
| **效果下降時** | 刪除不再代表品牌的舊範例 |

## 常見問題

### 設定風格要多久？

首次設定約需 10-15 分鐘（準備範例 + 等待分析）。設定完成後就會自動應用。

### 風格設定會影響已生成的文章嗎？

不會。風格設定只影響之後生成的新內容。已生成的文章保持不變。

### 可以完全移除品牌風格嗎？

可以。在風格設定頁面點擊「清除風格設定」，之後會使用預設風格。

### 不同類型文章適合不同風格怎麼辦？

可以在生成時手動選擇風格。例如新聞類內容選「客觀中立」，教學類內容使用品牌風格。
