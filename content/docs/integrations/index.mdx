## 為什麼需要整合？

Otlex 透過整合外部服務來獲取更完整的數據，提供更精準的分析和建議。

| 整合效益 | 說明 |
|---------|------|
| **真實數據** | 取得實際的搜尋排名和流量數據，而非估算值 |
| **全面分析** | 結合多種數據源，提供 360 度視角 |
| **自動化** | 直接發布內容到你的平台，無需手動複製 |
| **效率提升** | 減少在不同工具間切換的時間 |

## 支援的整合服務

### 數據整合

| 服務 | 功能 | 重要性 |
|-----|------|--------|
| **Google Search Console** | 搜尋效能（查詢/頁面）的點擊、曝光、CTR、平均排名 | ⭐⭐⭐ 必要 |
| **Google Analytics 4** | 流量分析、用戶行為、轉換追蹤 | ⭐⭐ 強烈建議 |

### 內容發布整合

| 平台 | 狀態 | 說明 |
|-----|------|------|
| **WordPress** | ✅ 基本發佈 | 可建立草稿/發布（依站點設定）；進階欄位（分類/標籤/特色圖片/SEO 外掛欄位）目前不在支援範圍 |
| **Shopify** | ✅ 基本發佈 | 發佈為部落格文章（基本欄位） |
| **Webflow** | 🚧 Beta | 發佈 CMS Item（需 token + collectionId + 欄位對應） |
| **Notion** | 🚧 預設臨時匯出 | 預設提供 Markdown/HTML 臨時匯出；啟用 API 模式後可建立 Notion Page/Database item（目前以純文字段落為主） |
| **Google Docs** | 🚧 預設臨時匯出 | 預設提供 Markdown/HTML 臨時匯出；啟用 API 模式後可建立 Google 文件（目前以純文字寫入為主） |

## 整合優先順序建議

### 第一步：連接 Google Search Console

GSC 是 **最重要** 的整合，建議在使用 Otlex 之前優先完成：

- 提供真實的搜尋排名數據
- 識別機會關鍵字
- 追蹤 SEO 成效變化
- 分析點擊率和曝光

沒有 GSC 整合，許多功能的效果會大打折扣。

### 第二步：連接 Google Analytics 4

GA4 整合可以幫助你：

- 了解實際流量
- 分析用戶行為
- 追蹤轉換目標
- 結合 SEO 與業務數據

### 第三步：設定內容發布整合

如果你使用 AI 內容生成功能，建議設定 CMS 整合：

- 直接在文章頁使用「發佈／匯出」輸出到目的地（依整合狀態決定可用項目）
- 省去手動複製貼上（或改成一鍵產出可複製的 HTML/Markdown）
- Notion / Google Docs 的「格式保留」依模式不同：臨時匯出（你自行貼上）或 API 寫入（目前以純文字為主）

## 整合管理

### 查看整合狀態

1. 進入「設定」>「整合」
2. 查看已連接的服務
3. 綠色勾號表示連接正常
4. 紅色警告表示需要處理

### 整合狀態說明

| 狀態 | 說明 | 行動 |
|-----|------|------|
| ✅ **已連接** | 整合正常運作 | 無需處理 |
| ⚠️ **需要重新授權** | Token 過期 | 點擊「重新連接」 |
| 🔴 **連接失敗** | 授權被撤銷或有錯誤 | 重新連接 |
| ⏳ **同步中** | 正在同步數據 | 等待完成 |

### 資料同步方式與延遲

| 服務 | 觸發方式 | 數據延遲 |
|-----|---------|---------|
| Google Search Console | 在「設定 > 整合」點擊同步（會排程背景任務）；部分功能頁也會提供「立即同步」入口 | 2-3 天（Google 限制） |
| Google Analytics 4 | 在「設定 > 整合」點擊同步（會排程背景任務） | 24-48 小時（GA4 處理延遲） |
| CMS 發佈 | 你在文章頁按下「發佈／匯出」時才會觸發 | 無 |

## 權限與隱私

### 我們只要求最小必要權限

| 服務 | 權限範圍 |
|-----|---------|
| GSC | 唯讀：搜尋數據、網站列表 |
| GA4 | 唯讀：分析數據 |
| Notion | 讀寫：你分享的頁面/資料庫 |
| Google Docs | 讀寫：建立和編輯文件 |
| WordPress | 讀寫：文章發布 |

### 資料安全保證

- **加密存儲**：所有 Token 會加密後存放
- **最小權限**：只要求必要的權限範圍
- **用途限定**：你的整合數據僅用於 Otlex 平台功能（AI 功能會將內容傳送至 AI 模型供應商處理，詳見服務條款）
- **可隨時撤銷**：你可以隨時中斷連接

## 常見問題

### 必須連接所有整合嗎？

不必。但強烈建議至少連接 Google Search Console，這是獲得最佳體驗的關鍵。

### 整合會存取我的其他數據嗎？

不會。每個整合只存取特定、必要的數據。例如：
- GSC 整合不會存取你的 Gmail
- GA4 整合不會存取你的 Google Drive

### 可以同時連接多個 GSC 資源嗎？

每個品牌只能連接一個 GSC 資源。如果你有多個網站，請建立多個品牌。

### 整合失效怎麼辦？

1. 檢查是否在 Google 帳戶中撤銷了授權
2. 嘗試中斷連接後重新連接
3. 確認 GSC/GA4 資源仍存在
4. 聯繫客服尋求協助
