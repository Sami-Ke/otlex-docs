## 你可以用它做什麼

關鍵字研究工具幫助你找出值得優化的關鍵字：

- **發現機會關鍵字**：找出有流量但排名不佳的關鍵字
- **擴展內容主題**：從種子詞發現相關的長尾關鍵字
- **找出內容缺口**：了解哪些搜尋需求還沒被滿足
- **分析競爭對手**：知道對手經營哪些你沒有的關鍵字

<Callout type="info">
  本工具以 **GSC 資料為核心**，不是 Ahrefs/SEMrush 等級的大型關鍵字資料庫。適合用於「已有一定流量的網站」找出優化機會，而非從零開始的大規模關鍵字調研。
</Callout>

## 快速開始

1. **確認 GSC 已連接**：進入「設定 > 整合」檢查 GSC 狀態
2. **進入關鍵字研究**：從側邊欄「工具 > 關鍵字研究」進入
3. **從「機會關鍵字」開始**：這是最能快速產生價值的分頁
4. **選出要優化的關鍵字**：根據分析結果決定行動

## 前置條件

| 條件 | 必要性 | 說明 |
|------|--------|------|
| GSC 已連接 | **必要** | 所有分析都基於真實搜尋資料 |
| GSC 有足夠資料 | 建議 | 至少 30 天的搜尋資料效果較佳 |
| 已完成內容爬取 | 建議 | 內容缺口分析需要頁面內容 |

## 四大功能分頁

### 機會關鍵字（推薦先看）

系統自動分析 GSC 資料，把關鍵字分成三類：

| 分類 | 定義 | 建議行動 |
|------|------|----------|
| **快速致勝** | 排名 11-20、曝光量高 | 稍微優化就能上首頁，優先處理 |
| **潛力成長** | 排名 21-50、曝光量中等 | 需要更多優化，但有潛力 |
| **需要新內容** | 有曝光但沒有對應頁面 | 考慮建立新內容 |

**如何使用**：

1. 切換到「機會關鍵字」分頁
2. 系統自動載入 GSC 資料並分類
3. 從「快速致勝」開始，這些最容易產生效果
4. 點擊關鍵字可查看對應的 landing page
5. 決定要優化哪些，加入排名追蹤

### 關鍵字建議

輸入種子關鍵字，獲得相關的延伸詞彙：

| 建議來源 | 說明 |
|----------|------|
| GSC 相關查詢 | 使用者實際搜尋到你網站的類似詞 |
| 語意延伸 | AI 產生的相關主題和長尾詞 |
| 搜尋變體 | 不同說法、同義詞、拼寫變化 |

**如何使用**：

1. 輸入 1-5 個種子關鍵字
2. 點擊「生成建議」
3. 檢視結果，勾選有興趣的關鍵字
4. 可將選中的關鍵字加入排名追蹤或匯出

<Callout type="info">
  建議功能會消耗 AI 額度，額度顯示在頁面右上角。
</Callout>

### 內容缺口

分析你的網站尚未覆蓋的搜尋需求：

| 缺口類型 | 說明 | 建議 |
|----------|------|------|
| **主題缺口** | 有搜尋量的主題，你完全沒有內容 | 建立新頁面 |
| **深度缺口** | 你有內容但不夠深入，對手比你完整 | 擴充既有內容 |
| **格式缺口** | 缺少特定格式（FAQ、教學、比較） | 調整內容格式 |

**如何使用**：

1. 系統自動比對你的內容與 GSC 查詢
2. 按「潛在價值」排序（曝光量 × 缺口程度）
3. 檢視每個缺口的詳細說明
4. 決定要填補哪些缺口

### 競品分析

比較你與競爭對手的關鍵字策略：

**輸入**：競爭對手的網站 URL

**輸出**：

| 分析面向 | 說明 | 如何利用 |
|----------|------|----------|
| **共同關鍵字** | 雙方都有排名 | 確保你的排名不輸對手 |
| **對手獨有** | 對手有、你沒有 | 評估是否要進攻 |
| **你的優勢** | 你有、對手沒有 | 繼續保持領先 |
| **排名比較** | 同一關鍵字的排名差異 | 找出需要加強的地方 |

<Callout type="warning">
  競品分析需要外部 API 資料，可能有額度限制或資料延遲。實際可用性依方案而異。
</Callout>

## 關鍵指標說明

### 搜尋量（Search Volume）

| 等級 | 月搜尋量 | 特性 |
|------|----------|------|
| 高 | > 1000 | 競爭激烈，但流量潛力大 |
| 中 | 100-1000 | 適合中小型網站經營 |
| 低 | < 100 | 競爭小，適合長尾策略 |

### 競爭度（Competition）

| 等級 | 說明 | 建議 |
|------|------|------|
| 低 | 排名靠前的頁面權重不高 | 新網站也有機會 |
| 中 | 需要一定的內容品質和連結 | 需要投入資源 |
| 高 | 大品牌和權威網站佔據 | 考慮換個切入角度 |

### 排名潛力評分

系統綜合以下因素計算：

- 目前排名位置
- 你的頁面與關鍵字的相關度
- 競爭對手的強度
- 你的網站權重

分數越高 = 越值得投入資源優化。

## 資料來源與更新頻率

| 資料類型 | 來源 | 更新頻率 |
|----------|------|----------|
| 你的關鍵字排名 | Google Search Console | GSC 同步時（2-3 天延遲） |
| 曝光量/點擊數 | Google Search Console | GSC 同步時（2-3 天延遲） |
| 搜尋量估算 | 外部 API | 即時查詢 |
| 關鍵字建議 | GSC + AI | 即時生成 |
| 競品資料 | 外部 API | 即時查詢 |

## 額度說明

| 功能 | 額度消耗 | 說明 |
|------|----------|------|
| 機會關鍵字 | 無 | 純 GSC 資料分析 |
| 關鍵字建議 | AI 額度 | 每次生成消耗點數 |
| 內容缺口 | 無 | 純內部資料分析 |
| 競品分析 | API 額度 | 依外部 API 計費 |

額度狀態顯示在頁面右上角，額度用完會有提示。

## 與其他功能整合

| 整合功能 | 如何搭配 |
|----------|----------|
| **排名追蹤** | 將研究到的關鍵字加入追蹤，持續監控 |
| **AI 內容生成** | 用關鍵字作為文章主題或大綱 |
| **SEO 審查** | 檢查頁面是否有針對關鍵字優化 |

## 常見問題

### 機會關鍵字是空的

**可能原因**：

| 原因 | 解法 |
|------|------|
| GSC 尚未連接 | 完成 GSC 授權 |
| GSC 資料不足 | 等待累積更多搜尋資料（建議至少 30 天） |
| 網站太新 | 新網站需要時間被 Google 收錄 |
| 資料同步中 | 等待同步完成，通常幾分鐘內 |

### 搜尋量和其他工具不同

這是正常的，原因：

- 不同工具使用不同資料來源
- 搜尋量是估算值，本身就有誤差
- 更新時間點不同導致差異

建議把搜尋量當作「相對參考」而非絕對數字。

### 競品分析查不到資料

**可能原因**：

- 輸入的 URL 格式不正確（需要完整網址）
- 該網站沒有足夠的公開排名資料
- API 額度已用完
- 該網站阻擋了資料抓取

### 如何選擇要優化的關鍵字？

優先順序建議：

1. **快速致勝類**：排名 11-20、曝光量高 → 最優先
2. **品牌相關詞**：確保品牌搜尋你是第一名
3. **高轉換意圖**：帶有購買/註冊意圖的詞
4. **低競爭長尾**：容易取得排名的利基詞

## 最佳實踐

### 每週關鍵字研究 SOP

建議每週花 30 分鐘：

1. **檢視機會關鍵字**：看有沒有新的快速致勝機會
2. **追蹤已優化的關鍵字**：確認上週優化的關鍵字有沒有進步
3. **更新內容缺口清單**：加入新發現的缺口到內容規劃
4. **監控競品動態**：看對手有沒有新的關鍵字布局

### 關鍵字到行動的轉換

```
發現機會關鍵字
├── 已有對應頁面？
│   ├── 是 → 優化該頁面（標題、內容、內連）
│   └── 否 → 建立新頁面
├── 加入排名追蹤
└── 一個月後評估效果
```

### 避免常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 只看搜尋量選詞 | 綜合考量競爭度和相關性 |
| 一次優化太多詞 | 每次聚焦 5-10 個關鍵字 |
| 忽略長尾關鍵字 | 長尾詞競爭小、轉換率高 |
| 不追蹤效果 | 加入排名追蹤，定期檢視 |
