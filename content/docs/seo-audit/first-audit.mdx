## 開始審查

<Steps>
  <Step>
    ### 進入 SEO Audit 頁面
    從左側選單點擊「SEO Audit」進入審查列表頁面。
  </Step>
  
  <Step>
    ### 輸入要審查的網址
    點擊「新增審查」，輸入你想要審查的頁面完整網址（包含 https://）。
  </Step>
  
  <Step>
    ### 等待審查完成
    系統會開始爬取和分析頁面，通常需要 30 秒到 2 分鐘。
  </Step>
  
  <Step>
    ### 查看審查報告
    審查完成後，點擊查看詳細報告和建議。
  </Step>
</Steps>

## 審查包含哪些項目？

### Technical SEO

- Title Tag 長度與格式
- Meta Description 存在與長度
- Canonical URL 設定
- Heading 結構 (H1-H6)
- 圖片 Alt 文字
- 內部連結數量

### Content 分析

- 內容長度與深度
- 關鍵字使用
- 可讀性評估
- 內容結構

### AEO (AI 搜尋優化)

- FAQ Schema 存在
- 問答格式內容
- 結構化數據完整性
- AI 爬蟲友善度

### 競品比較

- 同類頁面 Schema 對比
- 內容深度差異
- 結構差異分析

## 常見審查失敗原因

| 原因 | 說明 | 解決方式 |
|-----|------|---------|
| 登入牆 | 頁面需要登入才能存取 | 確保頁面對外公開 |
| CAPTCHA | 網站有驗證碼保護 | 將爬蟲 IP 加入白名單 |
| JavaScript 渲染 | 內容完全靠 JS 渲染 | 系統會嘗試 JS 渲染 |
| 網站阻擋 | 網站封鎖爬蟲 | 檢查 robots.txt 設定 |
