## 建議優先級

每個建議都有優先級標示，幫助你決定處理順序：

### 🔴 高優先 (Critical)

這些問題會嚴重影響 SEO 效果，應該**立即處理**：

- 缺少 Title Tag
- 缺少 Meta Description
- H1 標籤問題
- Canonical URL 設定錯誤
- 重大結構化數據錯誤

### 🟡 中優先 (Important)

這些問題有明顯改善空間，建議**盡快處理**：

- Title 或 Description 過長/過短
- 缺少圖片 Alt 文字
- Heading 結構不完整
- 缺少 FAQ Schema

### 🟢 低優先 (Nice to have)

這些是進階優化，**有時間再處理**：

- 結構化數據擴充
- 內容深度建議
- 進階 Schema 類型

## 建議類別詳解

### Technical SEO 建議

| 建議 | 說明 | 如何修復 |
|-----|------|---------|
| 新增 Meta Description | 頁面缺少描述 | 在 CMS 或 HTML 中新增 |
| 修正 Title 長度 | 標題過長會被截斷 | 控制在 60 字元內 |
| 設定 Canonical URL | 避免重複內容問題 | 在 head 中新增 canonical |
| 修復 H1 結構 | 應該只有一個 H1 | 調整 HTML 結構 |

### AEO 建議

| 建議 | 說明 | 如何修復 |
|-----|------|---------|
| 新增 FAQ Schema | AI 搜尋引擎偏好 FAQ | 使用 JSON-LD 新增 |
| 優化問答格式 | 內容以問答形式呈現 | 改寫內容結構 |
| 新增 llms.txt | 引導 AI 爬蟲 | 在根目錄新增檔案 |

### Content 建議

| 建議 | 說明 | 如何修復 |
|-----|------|---------|
| 增加內容深度 | 內容可能過於簡短 | 擴充相關內容 |
| 改善可讀性 | 段落過長或結構不清 | 分段並加入小標題 |
| 新增內部連結 | 連結到相關頁面 | 在內容中加入連結 |

## 可執行建議

部分建議提供「AI 協助執行」功能：

### 🤖 AI 生成 Alt 文字

系統會使用 Vision AI 分析圖片，自動生成描述性的 Alt 文字。

1. 點擊「AI 生成 Alt」
2. 預覽生成結果
3. 確認後複製到你的網站

### 🤖 AI 生成 Meta Description

根據頁面內容，AI 會撰寫符合 SEO 最佳實踐的描述。

## 追蹤建議處理進度

你可以標記每個建議的處理狀態：

- ⏳ 待處理
- 🔄 處理中
- ✅ 已完成
- ❌ 忽略（附原因）

下次審核時，系統會比對之前的建議，顯示哪些已改善、哪些仍未處理。
