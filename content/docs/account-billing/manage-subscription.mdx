## 在哪裡查看訂閱與計費？

進入「設定」>「方案與計費」頁面，可以看到：

| 資訊項目 | 說明 |
|---------|------|
| **目前方案名稱** | 你訂閱的方案（Free / Starter / Pro / Enterprise） |
| **計費週期** | 月付或年付 |
| **下次扣款日** | 下一個計費週期的開始日期 |
| **配額使用狀況** | 各項功能的用量 vs 總額度 |
| **帳單記錄** | 歷史付款記錄（依環境開放程度而定） |

### 配額使用儀表板

「方案與計費」頁面的配額區塊顯示：

- **SEO 審查次數**：本週期已使用 / 總額度
- **AI 生成點數**：已消耗 / 總額度
- **品牌數量**：已建立 / 上限
- **團隊成員**：目前人數 / 上限
- **下次重置日期**：月度配額何時重置

<Callout type="info">
  SEO 審查次數和 AI 生成點數是月度配額，每個計費週期會重置。品牌數量和成員上限是方案限制，不會每月重置。
</Callout>

## 升級方案

### 什麼時候該考慮升級？

以下情況建議升級：

| 症狀 | 對應需求 |
|------|---------|
| AI 點數不夠，每月底卡住 | 升級到更高點數配額的方案 |
| 審查次數不足以覆蓋所有頁面 | 升級 SEO 審查配額 |
| 需要管理更多網站（品牌） | 升級以解鎖更多品牌數量上限 |
| 要邀請更多成員 | 升級以增加成員上限 |
| 需要直接發布到 WordPress/Shopify | 升級至 Pro 以上（CMS 整合功能） |
| 需要 GA4 整合 | 升級至 Starter 以上 |

### 升級步驟

1. 進入「設定」>「方案與計費」
2. 點擊「升級方案」或「查看方案」
3. 選擇目標方案
4. 選擇計費週期（月付 / 年付）
5. 依環境導向至 **Stripe** 或 **TapPay** 完成結帳

**升級後的生效機制：**

- 功能與配額在付款完成後立即更新
- 若升級時仍在當期計費週期內，差額計算以結帳頁顯示為準
- 升級後的發票/收據可在帳單記錄查看

## 降級方案

### 降級前必讀

| 資源類型 | 降級後行為 |
|---------|-----------|
| **超出新方案的品牌數量** | 現有資料保留，但無法新增品牌 |
| **超出新方案的成員上限** | 現有成員暫時保留，新邀請被封鎖 |
| **進階功能（如 CMS 發布）** | 當期計費週期結束後停用 |
| **已生成的文章與內容** | 不會刪除，仍可查看與匯出 |
| **歷史審查記錄** | 完整保留 |
| **AI 點數餘額** | 若降級後配額較低，多餘點數不退還 |

### 如何降級

目前降級需要聯絡管理員或客服協助處理，不支援在 App 內自助操作。

## 計費週期與付款方式

### 支援的付款提供者

| 付款提供者 | 適用環境 | 支援付款方式 |
|-----------|---------|-------------|
| **Stripe** | 全球（依部署環境） | 信用卡（Visa / Mastercard / Amex 等） |
| **TapPay** | 台灣（依部署環境） | 台灣信用卡 |

你的帳戶使用哪個付款提供者，取決於你所在的部署環境，以「方案與計費」頁面顯示的結帳入口為準。

### 更新付款方式

- 若 App 內有「管理付款方式」入口，可直接操作
- 若無入口，請聯絡管理員或客服協助更換信用卡

<Callout type="warning">
  付款失敗（信用卡過期、餘額不足、卡片被凍結）可能導致訂閱中斷。請確保付款方式保持有效，並注意信用卡到期日。
</Callout>

## 帳單與發票

### 查看帳單記錄

1. 進入「設定」>「方案與計費」
2. 找到「帳單記錄」或「發票」區塊
3. 可查看每次扣款的金額、日期與狀態

### 下載發票

- 若帳單記錄頁面有「下載」按鈕，可直接下載 PDF 格式發票
- 若無自助入口，請聯絡客服提供發票/收據

### 發票資訊設定

若需在發票上顯示公司名稱、統一編號（台灣）或公司地址：
- 在首次付款的結帳流程中填寫計費資訊
- 若需修改已開立發票上的資訊，請聯絡客服

## 取消訂閱

### 取消前的注意事項

| 取消後的影響 | 說明 |
|------------|------|
| **服務持續至期末** | 取消後，當期服務仍繼續至計費週期結束 |
| **到期後降回 Free** | 到期後自動降至 Free 方案功能範圍 |
| **資料不立即刪除** | 帳號內容在取消後仍可存取（以平台政策為準） |
| **AI 點數清零** | 未使用的 AI 點數不退還 |
| **重訂不受影響** | 取消後可隨時重新訂閱 |

### 如何取消

目前取消訂閱需聯絡管理員或客服協助：
1. 在 App 內使用聯絡客服功能，或發送電子郵件至客服信箱
2. 說明你的帳號 Email 和取消需求
3. 取消確認後，服務持續到當期計費週期結束

## 自備 API Key（Bring Your Own Key）

若你的方案支援「自備 API Key（BYOK）」，可以使用自己的 AI Provider API Key 生成內容，不消耗 Otlex 的 AI 點數配額。

### 設定方式

1. 進入「設定」>「AI 設定」
2. 找到「AI 提供者」或「自備 API Key」區塊
3. 輸入你的 API Key（OpenAI 等主流提供者）
4. 儲存設定後，後續生成會使用你的 Key

### BYOK 適合的場景

| 場景 | 建議做法 |
|------|---------|
| 每月大量生成（100+ 篇文章） | BYOK 通常比方案配額更划算 |
| 需要使用特定 AI 模型 | 用 BYOK 指定偏好的模型版本 |
| 一般使用量 | Otlex 內建點數通常已足夠 |

<Callout type="info">
  使用 BYOK 時，AI 生成費用由你的 API Key 帳戶支付，與 Otlex 訂閱費用分開計算。請確保 API Key 有足夠餘額，否則生成任務會失敗。
</Callout>

## 常見問題

### 付款失敗會怎樣？

系統通常會在幾天內自動重試付款。若多次失敗，訂閱可能暫停，你會收到通知信提醒更新付款方式。在問題解決前，高於 Free 方案的功能可能暫時無法使用。

### 年付 vs 月付哪個更划算？

年付通常有折扣（以購買頁顯示的折扣率為準），適合已確定長期使用的團隊。月付靈活性高，適合還在評估期的使用者。

### 可以暫停訂閱嗎？

目前不支援暫停訂閱。若需暫時停用，可以選擇取消後在需要時重新訂閱。

### 退款政策是什麼？

退款政策以購買時的條款和結帳頁顯示為準。如有疑問請聯絡客服說明情況。

### 配額什麼時候重置？

月度配額（AI 點數、SEO 審查次數）在每個計費週期開始時重置，以你的訂閱開始日為基準（不一定是每月1日）。例如：若你在每月 15 日訂閱，重置日就是每月 15 日。

### 升級後舊的點數怎麼算？

升級後通常會立即獲得新方案的完整配額，舊方案的剩餘點數情況以結帳頁顯示為準。
