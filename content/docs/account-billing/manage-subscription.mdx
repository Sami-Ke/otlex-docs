## 查看目前方案

### 進入計費頁面

1. 點擊左下角的帳號頭像
2. 選擇「設定」
3. 點擊「方案與計費」

### 方案資訊面板

| 顯示項目 | 說明 |
|---------|------|
| **目前方案** | Starter / Pro / Business / Enterprise |
| **計費週期** | 月繳 / 年繳 |
| **下次扣款日** | 下一次自動扣款的日期 |
| **本期結束日** | 目前計費週期結束日期 |
| **本月費用** | 本計費週期的應付金額 |

### 用量統計

即時查看各項功能的使用情況：

| 項目 | 顯示格式 | 說明 |
|-----|---------|------|
| SEO 稽核 | 已用 / 配額（如 45/200） | 本月已執行的稽核次數 |
| AI 點數 | 已用 / 配額（如 120/200） | 本月已消耗的 AI 點數 |
| 品牌數 | 已建 / 上限（如 2/3） | 已建立的品牌數量 |
| 成員數 | 已加 / 上限（如 3/5） | 已加入的成員數量 |

<Callout type="info">
  用量達到 80% 時會收到提醒，100% 時該功能暫停直到下個週期或升級方案。
</Callout>

## 升級方案

### 升級步驟

1. 進入「設定」>「方案與計費」
2. 點擊「升級方案」或「變更方案」
3. 選擇目標方案（Pro / Business / Enterprise）
4. 選擇計費週期（月繳 / 年繳）
5. 確認付款資訊
6. 點擊「確認升級」

### 升級生效時間

| 項目 | 說明 |
|-----|------|
| **功能開放** | 立即生效 |
| **配額調整** | 立即按新方案配額 |
| **計費方式** | 按比例計算本月差額 |

### 按比例計費範例

假設你在月中（第 15 天）從 Pro ($49) 升級到 Business ($149)：

```
本月已付：$49（Pro 方案）
剩餘天數：15 天（本月共 30 天）
升級差額：($149 - $49) × (15/30) = $50
本次收取：$50
下月開始：每月 $149
```

### 從月繳改年繳

1. 在計費頁面點擊「切換為年繳」
2. 系統會計算優惠後的年費
3. 扣除本月已付費用（按比例）
4. 支付差額

## 降級方案

### 降級步驟

1. 進入「設定」>「方案與計費」
2. 點擊「變更方案」
3. 選擇較低的方案
4. 確認降級影響
5. 點擊「確認降級」

### 降級生效時間

| 項目 | 說明 |
|-----|------|
| **生效日期** | 下一個計費週期開始時 |
| **本週期** | 維持原方案所有功能 |
| **已付費用** | 不退款，用到週期結束 |

### 降級前的必要處理

如果新方案的限制低於目前使用量，降級前需要處理：

| 項目 | 處理方式 | 期限 |
|-----|---------|------|
| **品牌數超過上限** | 刪除或封存多餘品牌 | 降級前 |
| **成員數超過上限** | 移除多餘成員 | 降級前 |
| **已排程的稽核** | 自動取消超出配額的排程 | 自動處理 |

### 降級後的資料處理

| 資料類型 | 處理方式 |
|---------|---------|
| 稽核歷史 | 超過新方案保留期的會被刪除 |
| AI 生成內容 | 保留，但無法使用進階編輯功能 |
| 整合設定 | 不支援的整合會被停用 |
| 報表 | 無法再匯出進階報表 |

## 取消訂閱

### 取消步驟

1. 進入「設定」>「方案與計費」
2. 滾動到頁面最下方
3. 點擊「取消訂閱」
4. 選擇取消原因（選填）
5. 確認取消

### 取消後的處理

| 項目 | 說明 |
|-----|------|
| **可用期限** | 到目前計費週期結束 |
| **週期結束後** | 自動降級為 Starter（免費） |
| **資料保留** | 保留 30 天 |
| **恢復方式** | 30 天內重新訂閱可恢復資料 |

### 取消年繳方案

年繳方案取消政策：

| 時間點 | 退款政策 |
|-------|---------|
| 購買後 30 天內 | 全額退款（扣除已使用的按日費用） |
| 購買後 30 天後 | 不退款，可用到年度結束 |

## 查看帳單

### 帳單歷史

1. 進入「設定」>「方案與計費」
2. 點擊「帳單歷史」
3. 查看所有歷史帳單

### 帳單列表顯示

| 欄位 | 說明 |
|-----|------|
| 帳單日期 | 扣款日期 |
| 金額 | 扣款金額 |
| 狀態 | 已付款 / 待付款 / 失敗 |
| 發票 | 下載連結 |

### 發票內容

每張發票包含：

- 發票編號（可供報帳）
- 計費週期
- 方案名稱
- 收費項目明細
- 付款方式（末四碼）
- 統一編號（如有設定）

### 下載發票

1. 在帳單歷史中找到目標帳單
2. 點擊「下載發票」
3. 選擇格式（PDF）
4. 發票自動下載

## 管理付款方式

### 查看付款方式

1. 進入「設定」>「方案與計費」
2. 點擊「付款方式」
3. 查看已儲存的付款方式

### 新增付款方式

1. 點擊「新增付款方式」
2. 選擇類型（信用卡）
3. 輸入卡片資訊：
   - 卡號
   - 有效期限
   - 安全碼 (CVV)
   - 持卡人姓名
4. 選擇是否設為預設
5. 點擊「新增」

<Callout type="info">
  卡片資訊由 Stripe 安全處理，Otlex 不會儲存完整卡號。
</Callout>

### 變更預設付款方式

1. 在付款方式列表中找到目標卡片
2. 點擊卡片右側的選單
3. 選擇「設為預設」

### 移除付款方式

1. 找到要移除的付款方式
2. 點擊右側選單
3. 選擇「移除」

**注意**：
- 必須至少保留一個有效付款方式
- 無法移除預設付款方式（需先設定新的預設）

## 付款問題處理

### 扣款失敗

如果自動扣款失敗：

| 階段 | 處理 | 帳號影響 |
|-----|------|---------|
| **首次失敗** | 3 天後自動重試 | 正常使用 |
| **二次失敗** | Email 通知 + 7 天後重試 | 正常使用 |
| **三次失敗** | 帳號降級為 Starter | 付費功能停用 |

### 處理扣款失敗

1. 收到通知後進入「付款方式」
2. 確認卡片：
   - 是否過期
   - 是否有足夠額度
   - 是否被銀行阻擋
3. 更新付款資訊
4. 手動重新扣款（選用）

### 手動重新扣款

如果不想等待自動重試：

1. 進入「帳單歷史」
2. 找到失敗的帳單
3. 點擊「立即付款」
4. 選擇付款方式
5. 完成付款

## 統編發票設定

### 適用方案

Business 和 Enterprise 方案支援開立統一編號發票。

### 設定統編

1. 進入「設定」>「方案與計費」
2. 點擊「帳單設定」
3. 填寫：
   - 統一編號
   - 公司名稱
   - 帳單地址
4. 儲存設定

### 發票開立

設定後，所有新發票會自動包含統編資訊。已開立的發票無法更改。

## 常見問題

### 可以暫停訂閱嗎？

目前不支援暫停。你可以：
1. 降級到 Starter（免費），之後再升級
2. 取消訂閱，30 天內重新訂閱可恢復資料

### 升級後配額立即重置嗎？

是的。升級後配額立即調整為新方案的配額，本月已使用的會保留計算。

例如：本月已使用 45 次稽核，從 Starter (50 次) 升級到 Pro (200 次)，升級後剩餘 155 次可用。

### 年繳可以換成月繳嗎？

可以，但會在年度結束後生效。年度期間無法中途改為月繳。

### 支援哪些幣別？

目前只支援美金 (USD) 計價。帳單金額會以美金顯示，實際扣款金額依發卡銀行匯率換算。

### 可以申請退款嗎？

| 情況 | 退款政策 |
|-----|---------|
| 年繳 30 天內 | 可按比例退款 |
| 年繳 30 天後 | 不退款 |
| 月繳 | 當月不退款 |
| 重複扣款 | 聯繫客服處理 |

如有特殊情況，請聯繫 [support@otlex.com](mailto:support@otlex.com)。
