## 方案架構概覽

Otlex 提供多層級的方案架構，滿足不同規模團隊的需求：

| 層級 | 適用對象 | 主要特色 |
|------|----------|----------|
| **Free** | 個人評估試用 | 基本功能體驗，有限配額，適合評估是否符合需求 |
| **Starter** | 小型團隊、個人站長 | 核心功能完整，支援 GA4 整合與自備 API Key |
| **Pro** | 成長中的行銷團隊 | 進階功能全開，CMS 直接發布，更高配額 |
| **Enterprise** | 代理商、大型企業 | 客製化方案、專屬支援、無限制配額 |

## 如何選擇適合的方案？

### Free 方案適合你，如果你...

- 剛開始評估 Otlex 是否符合需求
- 只有一個小型網站，使用量需求低
- 不需要定期自動化執行分析

### Starter 方案適合你，如果你...

- 有穩定的 SEO 優化工作需求
- 需要 Google Analytics 整合查看完整流量
- 計畫定期使用 AI 文章生成（並考慮自備 API Key 節省成本）
- 有小型團隊需要協作

### Pro 方案適合你，如果你...

- 需要直接發布到 WordPress、Webflow 等 CMS（不想手動複製貼上）
- 管理多個品牌 / 網站
- 有較大的內容產出量（文章、社群貼文）
- 需要 HTTP API 整合到自訂系統

### Enterprise 方案適合你，如果你...

- 是數位行銷代理商，管理多個客戶
- 需要 API 存取程式化整合 Otlex
- 需要 SSO 統一帳號管理
- 需要 SLA 保證或客製化功能

## 配額與額度詳解

### 配額類型說明

| 配額類型 | 計算方式 | 重置週期 | 說明 |
|---------|---------|---------|------|
| **SEO 審查次數** | 每次執行 = 消耗 1 次 | 每月（依計費週期） | 整站審查和單頁審查都會消耗 |
| **AI 生成點數** | 依文章長度/功能消耗 | 每月（依計費週期） | 文章生成、選取改寫、社群貼文等 |
| **品牌數量** | 建立即佔用 | 無重置（方案上限） | 刪除品牌可釋放名額 |
| **團隊成員** | 邀請並接受即佔用 | 無重置（方案上限） | 移除成員可釋放名額 |

### 查看剩餘配額

進入「設定」>「方案與計費」可查看：
- 各項配額的已使用量 / 總額度
- 下次重置日期
- 使用趨勢圖

### 配額用完了怎麼辦？

| 情況 | 解決方式 |
|------|---------|
| AI 點數用完 | 等下個週期重置，或升級方案；若啟用 BYOK 則不受影響 |
| 審查次數用完 | 等下個週期重置，或升級方案 |
| 品牌數量達上限 | 刪除不用的品牌，或升級方案 |
| 成員數達上限 | 移除不需要的成員，或升級方案 |

## 自備 API Key（Bring Your Own Key）

Starter 以上方案支援「自備 API Key」功能，讓你使用自己的 OpenAI（或其他支援提供者）API Key：

### BYOK 的好處

| 好處 | 說明 |
|------|------|
| **不消耗 Otlex 點數** | AI 生成費用由你的 API 帳戶支付 |
| **使用量彈性** | 不受 Otlex 點數配額限制 |
| **成本可能更低** | 大量使用時，直接付 OpenAI 費用可能比升級方案划算 |

### 設定方式

1. 進入「設定」>「AI 設定」
2. 輸入你的 API Key
3. 儲存後，後續生成會優先使用你的 Key

<Callout type="info">
  使用 BYOK 時，如果 API Key 額度不足或無效，AI 功能會回退到使用 Otlex 內建點數（若有餘額）。請確保 API Key 有足夠餘額。
</Callout>

## 升級與降級

### 升級流程

1. 進入「設定」>「方案與計費」
2. 點擊「升級方案」
3. 選擇目標方案和計費週期（月付 / 年付）
4. 完成 Stripe 或 TapPay 結帳
5. 升級後功能立即生效

**年付優惠**：年付通常有折扣，以購買頁顯示的折扣率為準。

### 降級注意事項

降級需來信 [support@ethorx.com](mailto:support@ethorx.com) 協助。降級後：
- 超出新方案限制的品牌/成員數據保留但無法新增
- 進階功能在當期計費週期結束後停用
- 已生成的內容和歷史審查記錄不會刪除

## 常見問題

### 可以先免費試用嗎？

Free 方案提供基本功能的完整體驗（有配額限制），讓你評估是否符合需求。你不需要填寫信用卡就能開始使用基本功能。

### 年付和月付哪個比較划算？

年付通常提供 15-20% 的折扣（以購買頁顯示為準），適合已確定長期使用的團隊。月付靈活性高，適合還在評估期或業務有季節性的使用者。

### 可以在方案間隨時切換嗎？

升級可以隨時進行，立即生效。降級需來信 [support@ethorx.com](mailto:support@ethorx.com) 協助，通常在當期計費週期結束後生效。

### 代理商可以用一個帳號管理多個客戶嗎？

可以。你可以在同一個組織下建立多個品牌（每個客戶一個品牌），或者為每個客戶建立獨立的組織（需要多個帳號）。Pro 方案以上支援更多品牌數量，Enterprise 支援無限制。

### 自備 API Key 適合什麼情況？

如果你每月需要大量生成文章（超過方案內建的 AI 點數），自備 API Key 通常更划算，因為 OpenAI 的計費是按使用量計費，而不是固定月費。Starter 以上方案支援此功能。
