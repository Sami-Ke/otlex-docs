## 帳號架構說明

Otlex 使用三層組織架構：

```
帳號（User Account）
  └── 組織（Organization）
        ├── 品牌 A（Brand）
        ├── 品牌 B（Brand）
        └── 團隊成員（Members）
```

### 各層級說明

| 層級 | 說明 | 管理內容 |
|-----|------|---------|
| **帳號** | 你的個人登入帳號 | 個人資料、密碼、登入方式 |
| **組織** | 公司或團隊的最高層級 | 訂閱方案、計費、成員管理 |
| **品牌** | 代表一個網站/品牌 | SEO 數據、內容、整合設定 |

## 帳號管理

### 個人資料設定

在「帳號設定」中管理你的個人資料：

| 設定項目 | 說明 |
|---------|------|
| **顯示名稱** | 在系統中顯示的名稱 |
| **Email** | 登入用 Email（變更需驗證） |
| **頭像** | 上傳個人頭像 |
| **時區** | 報表和通知使用的時區 |
| **語言** | 介面語言偏好 |

### 安全設定

| 功能 | 說明 |
|-----|------|
| **變更密碼** | 定期更新密碼確保安全 |
| **兩步驟驗證** | 啟用 2FA 增加帳號安全性 |
| **登入記錄** | 查看最近的登入活動 |
| **已授權應用** | 管理已連接的第三方服務 |
| **登出所有裝置** | 從所有裝置強制登出 |

### 通知設定

自訂接收通知的方式：

| 通知類型 | Email | Dashboard | 預設 |
|---------|:-----:|:---------:|:----:|
| SEO 分數變化 | ✅ | ✅ | 開啟 |
| 排程稽核完成 | ✅ | ✅ | 開啟 |
| 帳單與付款 | ✅ | ✅ | 開啟 |
| 產品更新 | ✅ | ❌ | 開啟 |
| 行銷活動 | ✅ | ❌ | 關閉 |

## 組織管理

### 組織設定

組織擁有者和管理員可以管理：

| 設定 | 說明 |
|-----|------|
| **組織名稱** | 顯示在介面和發票上 |
| **組織 Logo** | 用於報表和匯出文件 |
| **統一編號** | 供開立發票用（選填） |
| **帳單地址** | 發票上的地址 |
| **組織時區** | 報表的預設時區 |

### 成員管理

管理組織內的團隊成員：

| 功能 | 說明 |
|-----|------|
| 邀請成員 | 透過 Email 邀請新成員 |
| 設定角色 | 指派 Owner/Admin/Member/Viewer 角色 |
| 調整權限 | 變更現有成員的角色 |
| 移除成員 | 從組織移除成員 |

詳細說明請參考 [邀請團隊成員](/docs/account-billing/invite-members)。

### 角色權限總覽

| 角色 | 權限範圍 |
|-----|---------|
| **Owner** | 完整權限，包含刪除組織、轉移所有權 |
| **Admin** | 管理成員、計費、整合，但無法刪除組織 |
| **Member** | 使用所有功能（稽核、生成、編輯），無法管理設定 |
| **Viewer** | 僅能查看數據和報表，無法執行任何操作 |

## 訂閱與計費

### 查看目前方案

1. 進入「設定」>「方案與計費」
2. 查看目前訂閱狀態：

| 資訊 | 說明 |
|-----|------|
| 目前方案 | Starter / Pro / Business / Enterprise |
| 計費週期 | 月繳或年繳 |
| 下次扣款日 | 下一次自動扣款的日期 |
| 用量統計 | SEO 稽核次數、AI 生成點數使用情況 |

### 用量監控

即時追蹤各項功能的使用量：

| 項目 | 顯示內容 |
|-----|---------|
| SEO 稽核 | 已使用 / 總配額 |
| AI 生成點數 | 已使用 / 總配額 |
| 品牌數 | 已建立 / 上限 |
| 成員數 | 已加入 / 上限 |
| 資料保留 | 歷史數據保留天數 |

用量接近上限時會收到提醒通知。

### 升級與降級

- **升級**：隨時升級，立即生效
- **降級**：下一個計費週期生效
- **取消**：可用到目前週期結束

詳細說明請參考 [管理訂閱](/docs/account-billing/manage-subscription)。

## 資料與隱私

### 資料保留

| 方案 | 資料保留期限 |
|-----|------------|
| Starter | 30 天 |
| Pro | 90 天 |
| Business | 1 年 |
| Enterprise | 永久或依合約 |

### 資料匯出

你隨時可以匯出你的資料：

- SEO 稽核報告
- 關鍵字追蹤數據
- AI 生成的內容
- 帳號設定

### 帳號刪除

如果要永久刪除帳號：

1. 進入「帳號設定」
2. 滾動到最下方
3. 點擊「刪除帳號」
4. 確認操作

<Callout type="warning">
  帳號刪除後，所有資料將永久刪除且無法恢復。建議先匯出重要資料。
</Callout>

## 常見問題

### 忘記密碼怎麼辦？

1. 在登入頁點擊「忘記密碼」
2. 輸入註冊 Email
3. 收取重設密碼信件
4. 設定新密碼

### 可以變更組織擁有者嗎？

可以。現任擁有者可以將所有權轉移給其他管理員：

1. 進入「設定」>「成員管理」
2. 找到要轉移的對象
3. 點擊「轉移所有權」
4. 確認轉移

### 帳號被鎖定怎麼辦？

連續登入失敗 5 次會暫時鎖定帳號。請等待 15 分鐘後重試，或使用「忘記密碼」重設。
