## 帳號管理

### 個人資料

在「帳號設定」中，你可以：

- 更新顯示名稱
- 變更 Email（需驗證）
- 設定通知偏好

### 安全設定

- 變更密碼
- 查看登入記錄

## 組織管理

### 成員管理

組織擁有者和管理員可以：

- 邀請新成員加入
- 設定成員角色權限
- 移除成員

### 角色權限

| 角色 | 說明 |
|-----|------|
| Owner | 組織擁有者，完整權限 |
| Admin | 管理員，可管理成員和設定 |
| Member | 成員，可使用所有功能 |
| Viewer | 檢視者，僅能查看數據 |

## 訂閱方案

### 查看目前方案

前往「設定」>「方案與計費」查看：

- 目前訂閱方案
- 用量統計
- 下次扣款日期

### 升級 / 降級

你可以隨時調整訂閱方案。變更將在下一個計費週期生效。
