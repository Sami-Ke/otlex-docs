## 開始審核

### 方式一：從 Dashboard 開始

1. 進入品牌 Dashboard
2. 在「快速操作」區塊點擊「執行 SEO 審核」
3. 輸入要審核的頁面網址
4. 點擊「開始審核」

### 方式二：從 SEO Audit 頁面

1. 從左側選單進入「SEO Audit」
2. 點擊「新增審核」
3. 輸入網址並開始

## 審核過程

審核通常需要 30 秒到 2 分鐘，取決於頁面複雜度。過程中 Otlex 會：

1. **爬取頁面** - 取得頁面完整 HTML
2. **分析結構** - 檢查 Schema、Meta Tags、內容結構
3. **AI 評估** - 使用 AI 分析內容品質與優化建議
4. **生成報告** - 產出評分與具體建議

<Callout type="warning">
  如果頁面有登入牆、CAPTCHA 或嚴重的 JavaScript 渲染問題，審核可能會失敗或結果不準確。系統會自動偵測這些情況並提醒你。
</Callout>

## 理解審核結果

### SEO 分數

0-100 的綜合評分，考量以下因素：

- Technical SEO（技術面）
- Content（內容品質）
- AEO（AI 搜尋優化）
- Schema（結構化數據）

### 四大類建議

| 類別 | 說明 | 範例 |
|-----|------|------|
| **AEO** | AI 搜尋引擎優化 | 新增 FAQ Schema、優化問答格式 |
| **Technical** | 技術層面問題 | 修復 Canonical、新增 Meta Description |
| **Content** | 內容品質改進 | 增加內容深度、優化標題結構 |
| **Competitor** | 競品對比分析 | 競品有的 Schema 你沒有 |

### 優先級標示

- 🔴 **高優先** - 嚴重影響 SEO，建議立即處理
- 🟡 **中優先** - 有改善空間，建議處理
- 🟢 **低優先** - 錦上添花，有時間再處理

## 執行建議

部分建議提供「一鍵執行」功能：

- **AI 生成 Alt 文字** - 自動為圖片生成描述性 Alt 文字
- **生成 Meta Description** - AI 撰寫頁面描述

其他建議需要你手動到 CMS 或網站後台修改。

## 保存與追蹤

每次審核結果會自動保存，你可以：

- 查看歷史審核記錄
- 比較不同時間的分數變化
- 追蹤建議的處理進度
