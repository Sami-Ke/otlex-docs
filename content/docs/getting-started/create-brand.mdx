## 什麼是品牌？

在 Otlex 中，**品牌 (Brand)** 是核心的工作單位，代表一個獨立的網站或產品線。所有的 SEO 分析、內容生成、數據追蹤都是以品牌為單位進行的。

### 品牌包含的資源

每個品牌都有獨立的：

| 資源類型 | 說明 |
|---------|------|
| 網站網址 | 你要優化的網站首頁 URL |
| 內容庫 | 該網站的所有已爬取頁面 |
| 稽核記錄 | SEO 稽核的歷史分數與建議 |
| 主題規劃 | AI 建議的內容主題與你的主題草稿 |
| 文章內容 | 使用 AI 生成或手動建立的文章 |
| 整合設定 | Google Search Console、Analytics 等連接 |
| 品牌風格 | 用於 AI 內容生成的品牌語調設定 |

### 何時需要建立多個品牌？

以下情況建議建立獨立的品牌：

- 你管理多個不同網域的網站
- 同一公司有官網和獨立部落格（不同子網域）
- 代理商為不同客戶管理 SEO

<Callout type="info">
  **子目錄 vs 子網域**：如果你的部落格是 `example.com/blog`（子目錄），通常歸入同一品牌即可。如果是 `blog.example.com`（子網域），建議建立獨立品牌。
</Callout>

## 建立品牌步驟

### 步驟 1：進入品牌管理

有兩種方式可以建立新品牌：

**方式一：從側邊欄**
1. 點擊左上角的品牌選擇器（顯示目前品牌名稱的地方）
2. 點擊「新增品牌」

**方式二：從設定頁面**
1. 進入左側選單「設定」>「品牌管理」
2. 點擊右上角「新增品牌」按鈕

### 步驟 2：填寫基本資訊

| 欄位 | 必填 | 說明 |
|-----|------|------|
| 品牌名稱 | ✅ | 在 Otlex 中顯示的名稱，例如「官網」或「產品部落格」 |
| 網站網址 | ✅ | 要優化的網站首頁 URL，例如 `https://example.com` |
| 品牌描述 | ❌ | 簡短描述這個品牌/網站的定位與目標受眾 |
| 品牌 Slug | ✅ | URL 友善的識別碼，自動從品牌名稱生成，可手動修改 |

<Callout type="warning">
  **網址格式注意事項**：
  - 請輸入完整網址，包含 `https://`
  - 不要包含結尾斜線，例如用 `https://example.com` 而非 `https://example.com/`
  - 如果網站有 www 和非 www 版本，請輸入主要使用的版本
</Callout>

### 步驟 3：設定品牌風格（選用）

品牌風格設定會影響 AI 內容生成的輸出。你可以稍後再設定，但建議在生成第一篇文章前完成。

#### 基本風格設定

| 設定項目 | 說明 |
|---------|------|
| 品牌語調 | 選擇品牌的溝通風格：專業、親切、幽默、權威等 |
| 目標受眾 | 描述你的主要讀者群體，例如「25-40歲的科技業行銷人員」 |
| 產業類別 | 選擇你的產業，幫助 AI 理解專業術語 |

#### 進階風格設定

| 設定項目 | 說明 |
|---------|------|
| 禁用詞彙 | 列出品牌絕對不使用的詞彙或表達方式 |
| 偏好詞彙 | 列出品牌偏好使用的特定用語 |
| 寫作範例 | 上傳或貼上品牌過去的文章作為風格參考 |

### 步驟 4：上傳品牌視覺（選用）

| 項目 | 用途 |
|-----|------|
| 品牌 Logo | 顯示在報表匯出、儀表板等位置 |
| 品牌主色 | 用於圖表配色、報表視覺 |

### 步驟 5：確認並建立

檢查所有資訊無誤後，點擊「建立品牌」按鈕。系統會：

1. 建立品牌並產生專屬的工作空間
2. 自動觸發首次網站爬取（可能需要幾分鐘）
3. 將你導向該品牌的 Dashboard

## 建立後的下一步

品牌建立完成後，建議按照以下順序進行設定：

1. **連接 Google Search Console** - 獲取真實的搜尋排名數據
2. **等待首次爬取完成** - 系統會自動爬取網站並建立內容庫
3. **執行第一次 SEO 稽核** - 了解網站的 SEO 健康度
4. **探索內容地圖** - 視覺化了解網站結構

## 管理現有品牌

### 切換品牌

點擊左上角的品牌選擇器，即可在不同品牌之間切換。

### 編輯品牌設定

進入「設定」>「品牌管理」，點擊要編輯的品牌，即可修改名稱、描述、風格設定等。

### 刪除品牌

<Callout type="error">
  刪除品牌會永久移除該品牌的所有數據，包含稽核記錄、內容、主題等。此操作無法復原。
</Callout>

如需刪除品牌，請至「設定」>「品牌管理」>「進階設定」>「刪除品牌」。

## 常見問題

### 可以建立幾個品牌？

品牌數量上限取決於你的訂閱方案。詳情請參考 [方案與定價](/docs/account-billing/plans)。

### 可以將品牌轉移到其他組織嗎？

目前不支援品牌跨組織轉移。如有需求，請聯繫客服協助處理。

### 建立品牌後可以修改網站網址嗎？

可以，但修改網址後：
- 原有的爬取內容將被清除
- 需要重新連接 Google Search Console
- 稽核歷史會保留但不再具有對比意義
