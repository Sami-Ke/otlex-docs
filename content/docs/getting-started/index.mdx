## 開始之前

在開始使用 Otlex 之前，請確認你已準備好以下項目：

### 必要條件

1. **一個網站**：你想要優化的網站網址（需為已上線且可公開訪問的網站）
2. **Otlex 帳號**：透過 Email 或 Google 登入註冊

### 建議準備（選用但強烈推薦）

3. **Google Search Console 存取權限**：用於獲取真實的搜尋數據與排名資訊
4. **Google Analytics 存取權限**：用於分析流量與使用者行為數據
5. **品牌相關資料**：品牌名稱、Logo、品牌色彩、目標受眾描述等

## Otlex 的核心概念

在開始操作之前，先了解 Otlex 的架構會幫助你更有效率地使用平台。

### 組織（Organization / 租戶 Tenant）

組織是 Otlex 的最高層級單位（在系統內也可能稱為 Tenant），通常對應到一間公司或團隊。組織層級管理：

- **成員與權限**：邀請團隊成員、設定角色權限
- **訂閱與計費**：管理訂閱方案、查看用量與帳單
- **AI 使用配額**：設定 AI 功能的使用上限
- **共用設定**：跨品牌共用的 AI Provider 設定

### 品牌 (Brand)

品牌代表一個獨立的網站或產品線。每個組織可以建立多個品牌，每個品牌擁有獨立的：

- 網站網址與內容庫
- SEO 審查記錄與分數歷史
- 主題規劃與文章內容
- Google Search Console / Analytics 整合
- 品牌風格設定（用於 AI 內容生成）

<Callout type="info">
  **舉例說明**：假設你的公司「ABC 科技」有一個官網和一個部落格，你可以建立一個組織「ABC 科技」，然後在下面建立兩個品牌：「ABC 官網」和「ABC Blog」。
</Callout>

## 設定流程總覽

完整的設定流程如下：

```
1. 建立組織 → 2. 建立品牌 → 3. 連接整合 → 4. 設定品牌風格 → 5. 開始使用
```

### 步驟 1：建立組織（若系統有引導）

首次登入 Otlex 時，系統通常會引導你建立組織。你只需要輸入：

- 組織名稱（公司或團隊名稱）
- 組織類型（公司、代理商、個人）

### 步驟 2：建立品牌

建立組織後，接著建立你的第一個品牌。詳細步驟請參考 [建立你的第一個品牌](/docs/getting-started/create-brand)。

### 步驟 3：連接整合

強烈建議連接 Google Search Console，這能讓 Otlex 取得：

- 關鍵字排名數據
- 曝光與點擊統計
- 查詢/頁面層級的搜尋效能資料（CTR、平均排名等）

詳細步驟請參考 [連接 Google Search Console](/docs/getting-started/connect-gsc)。

### 步驟 4：設定品牌風格

如果你計畫使用 AI 內容生成功能，建議先在「設定 > 品牌設定 > 寫作風格」建立 Style Profile（會根據已爬取的網站內容分析），常見包含：

- 品牌語調（專業、親切、幽默等）
- 目標受眾描述
- 禁用詞彙

這能確保 AI 生成的內容符合你的品牌調性。

### 步驟 5：開始使用

設定完成後，你可以：

- 執行第一次 SEO 審查
- 探索內容地圖
- 進入「主題」建立/生成主題，再到「內容生成」產生文章

## 新手建議的操作順序

如果你是第一次使用 Otlex，建議按照以下順序體驗各功能：

1. **執行 SEO 審查**：了解網站目前的 SEO 健康度
2. **查看內容地圖**：視覺化了解網站的內容結構
3. **使用主題（Topics）**：生成主題或做競品內容差距分析（Gap analysis）
4. **嘗試生成文章（Generation）**：產生文章草稿並在編輯器完成發佈/匯出
