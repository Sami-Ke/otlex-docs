## 什麼是 Otlex？

Otlex 是專為內容行銷團隊打造的一站式 AI SEO 平台。透過智慧化的內容審查、內容生成、排名追蹤、內容地圖視覺化、整合分析等功能，幫助你：

- **AI Search 審查**：深度分析網頁的 SEO 健康度，針對傳統搜尋引擎與 AI 搜尋引擎（如 ChatGPT、Perplexity）提供優化建議
- **自動化排程（Scheduler）**：定期執行 SEO 審查、網站爬蟲、內容重複檢測、競爭對手分析、分析同步、商家資訊提要（依方案/權限啟用），支援每日/每週/每月頻率
- **AI 內容生成**：基於品牌風格與關鍵字洞察，快速產出文章草稿與改寫版本，並支援發佈到 CMS 或匯出
- **內容地圖（Content Map）**：視覺化呈現網站的內容結構與內部連結關係，發現內容缺口與優化機會
- **整合與追蹤**：連接 Google Search Console / Google Analytics（GA4），並提供以 GSC 為主的排名與內容洞察
- **AI Bot Tracking**：追蹤 AI bots（例如 ChatGPT / Perplexity）對你網站的存取行為，補足傳統分析工具看不到的流量面向
- **AI 智慧助手**：App 內建 AI Chatbot，可即時回答你的 SEO 問題並協助操作

## 核心功能模組

### 策略儀表板

| 功能 | 說明 |
|------|------|
| 儀表板 | 營運數據總覽：流量趨勢、內容洞察、頁面類型分析、快速致勝建議、關鍵字機會矩陣、權威性快照、Schema 涵蓋矩陣 |
| 內容地圖 | 視覺化網站內容結構：多檢視模式、分析內部連結與主題群集（需 GSC + 爬蟲連接） |
| AI Search 審查 | 全面檢測網頁 SEO 問題，包含 AEO、技術 SEO、內容品質評分、競品對比 |
| 排名追蹤（以 GSC 為主） | 監控關鍵字排名變化、趨勢分析、排名警報、AI 搜尋影響分析（GSC 通常有 2–3 天延遲） |
| AI Readiness | 評估網站內容對 AI 搜尋引擎的友善程度，提供結構化資料分析與優化建議 |

### 內容工作流

| 功能 | 說明 |
|------|------|
| 主題（Topics） | 主題生成設定、已生成主題、內容策略 Tab、競品內容差距分析（Gap analysis） |
| 文章生成（Generation） | 以主題為中心建立生成條件，透過背景任務產生文章草稿；含生成設定、已生成文章、內容表現分析 |
| 文章編輯器 | 富文字/HTML 編輯、插入圖片、選取文字請 AI 改寫、內部連結建議、註解與協作、歷史版本 |
| 社群貼文 | 產生多平台社群貼文草稿（支援 UTM），在「社群運營」集中管理貼文、排程與平台帳號 |
| 自動發佈 | 設定自動發佈排程、CMS 目標地址整合（需在整合設定中啟用） |
| 發布與匯出 | WordPress / Webflow / Notion / Google Docs 等目的地（依設定啟用），Shopify UI 即將開放 |

### 專用工具

| 工具 | 說明 |
|------|------|
| FAQ Schema 工具 | 自動生成 FAQ 問答並匯出 JSON-LD 結構化資料；支援儲存與管理已生成的 FAQ |
| 內部連結工具 | 分析全站內容，建議最佳內部連結機會 |
| 關鍵字研究 | 以 GSC 資料與外部 API（DataForSEO）為基礎的關鍵字探索與洞察 |
| 重複內容偵測 | 檢測網站內是否存在重複或相似的頁面內容 |
| 競爭對手分析 | 深度分析競品網站的 SEO 策略、關鍵字涵蓋與內容差距 |
| 內容衰退檢測 | 識別流量或排名下降的既有內容，提供修復與更新建議 |
| GSC 資料探索器 | 深入探索 Google Search Console 原始數據，發現隱藏的搜尋機會 |
| 結構化資料分析 | 分析網站的 Schema Markup 實作狀態與涵蓋率 |
| 品牌提及追蹤 | 追蹤品牌在外部網站的提及與引用情況 |
| 商品目錄最佳化 | 針對電商網站的商品頁面進行 SEO 最佳化分析（依方案啟用） |
| 頁面類型管理 | 分類與管理網站頁面類型，調整各類型的 SEO 評分權重 |

### 成效追蹤

| 功能 | 說明 |
|------|------|
| 排名追蹤（以 GSC 為主） | 關鍵字追蹤、趨勢分析、排名警報、AI 搜尋影響分析（GSC 通常有 2–3 天延遲） |
| AI Readiness | 評估網站內容對 AI 搜尋引擎的友善程度，提供結構化資料分析與優化建議 |
| 內容機會（Opportunities） | 主題發現與內容差距分析，找出值得投資的新內容方向 |
| AI Bot Tracking | 追蹤 AI 爬蟲存取行為，分析 AI 搜尋對網站的影響 |
| 排程器（Scheduler） | 自動化排程：SEO 審查、網站爬蟲、重複內容偵測、競爭對手分析、分析同步、商家資訊提要 |

### 任務管理

| 功能 | 說明 |
|------|------|
| 任務管理 | 集中管理 SEO 優化任務與票券，統一追蹤待辦項目 |
