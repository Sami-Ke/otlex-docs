## 什麼是 Otlex？

Otlex 是專為內容行銷團隊打造的一站式 AI SEO 平台。透過智慧化的內容稽核、AI 寫作助手、內容地圖視覺化等功能，幫助你：

- **AI Search 稽核**：深度分析網頁的 SEO 健康度，針對傳統搜尋引擎與 AI 搜尋引擎（如 ChatGPT、Perplexity）提供優化建議
- **AI 內容生成**：基於品牌風格與目標關鍵字，快速產出高品質、SEO 優化的文章
- **內容地圖**：視覺化呈現網站的內容結構與內部連結關係，發現內容缺口與優化機會
- **排名追蹤**：監控目標關鍵字在 Google 搜尋結果中的排名變化
- **任務管理**：統一管理所有待辦事項，從稽核建議到內容排程一目了然

## 核心功能模組

### 策略儀表板

| 功能 | 說明 |
|------|------|
| 儀表板 | 營運數據總覽，快速掌握網站 SEO 現況 |
| 內容地圖 | 視覺化網站內容結構，分析內部連結與主題群集 |
| AI Search 稽核 | 全面檢測網頁 SEO 問題，包含 AEO、技術 SEO、內容品質評分 |
| 任務管理 | 集中管理稽核建議、內容任務、排程待辦 |

### 內容工作流

| 功能 | 說明 |
|------|------|
| 主題探索 | AI 根據品牌定位與競爭對手分析，建議高潛力內容主題 |
| 內容生成 | 輸入主題後自動產出完整文章，包含標題、大綱、內文 |
| 社群運營 | 管理社群貼文排程，追蹤 UTM 連結成效 |
| 排程器 | 統一檢視與管理所有已排程的內容發布時間 |

### 專用工具

| 功能 | 說明 |
|------|------|
| FAQ Schema 工具 | 自動生成 FAQ 問答並匯出 JSON-LD 結構化資料 |
| Internal Links 工具 | 分析全站內容，建議最佳內部連結機會 |
| 關鍵字研究 | 查詢關鍵字搜尋量、競爭度，發掘長尾關鍵字 |
| 反向連結 | 追蹤與管理網站的外部連結來源 |
| 重複內容偵測 | 檢測網站內是否存在重複或相似的頁面內容 |

### 成效追蹤

| 功能 | 說明 |
|------|------|
| 排名追蹤 | 每日監控目標關鍵字的 Google 排名變化 |
| AI Readiness | 評估網站內容對 AI 搜尋引擎的友善程度 |
