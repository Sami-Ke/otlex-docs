## 什麼是 Otlex？

Otlex 是專為內容行銷團隊打造的一站式 AI SEO 平台。透過智慧化的內容審查、內容生成、內容地圖視覺化、整合分析等功能，幫助你：

- **AI Search 審查**：深度分析網頁的 SEO 健康度，針對傳統搜尋引擎與 AI 搜尋引擎（如 ChatGPT、Perplexity）提供優化建議
- **自動化排程（Scheduler）**：定期執行 SEO 審查、網站爬蟲（內容快取更新）、內容重複檢測、競爭對手分析（功能依方案/權限啟用）
- **AI 內容生成**：基於品牌風格與關鍵字洞察，快速產出文章草稿與改寫版本
- **內容地圖（Content Map）**：視覺化呈現網站的內容結構與內部連結關係，發現內容缺口與優化機會
- **整合與追蹤**：連接 Google Search Console / Google Analytics（GA4），並提供（以 GSC 為主的）排名與內容洞察
- **AI Bot Tracking**：追蹤 AI bots（例如 ChatGPT / Perplexity）對你網站的存取行為，補足傳統分析工具看不到的流量面向

## 核心功能模組

### 策略儀表板

| 功能 | 說明 |
|------|------|
| 儀表板 | 營運數據總覽，快速掌握網站 SEO 現況 |
| 內容地圖 | 視覺化網站內容結構，分析內部連結與主題群集 |
| AI Search 審查 | 全面檢測網頁 SEO 問題，包含 AEO、技術 SEO、內容品質評分 |
| 排名追蹤（以 GSC 為主） | 監控關鍵字/頁面在 Google 搜尋中的表現變化（GSC 通常有 2–3 天延遲） |

### 內容工作流

| 功能 | 說明 |
|------|------|
| 主題（Topics） | 主題生成設定、已生成主題、競品內容差距分析（Gap analysis） |
| 文章生成（Generation） | 以主題為中心建立生成條件，透過背景任務產生文章草稿 |
| 社群貼文 | 產生多平台社群貼文草稿（支援 UTM），並在「社群運營」管理貼文、排程與平台帳號 |
| 發布與匯出 | WordPress / Shopify / Webflow / Notion / Google Docs / Webhook 等目的地（依設定啟用） |

### 專用工具

| 功能 | 說明 |
|------|------|
| FAQ Schema 工具 | 自動生成 FAQ 問答並匯出 JSON-LD 結構化資料 |
| Internal Links 工具 | 分析全站內容，建議最佳內部連結機會 |
| 關鍵字研究 | 以 GSC/既有資料為主的關鍵字探索與洞察（非 Ahrefs/SEMrush 等級大型資料庫） |
| 反向連結 | 反向連結頁面與基本檢視（非完整 backlinks 資料庫；GSC Links API 已停止） |
| 重複內容偵測 | 檢測網站內是否存在重複或相似的頁面內容 |

### 成效追蹤

| 功能 | 說明 |
|------|------|
| 排名追蹤（以 GSC 為主） | 以 GSC 同步資料為主（GSC 通常有 2–3 天延遲） |
| AI Readiness | 評估網站內容對 AI 搜尋引擎的友善程度 |
